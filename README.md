🚀 OIBSIP – Web Development Level 1 Projects

This repository contains three Web Development Level 1 projects completed as part of the OIBSIP Internship Program. Each project focuses on strengthening core front-end development skills using HTML5 and CSS3, with an emphasis on layout, responsiveness, and clean UI design.
 
📁 Project Structure
OIBSIP

🚀 Projects Overview

1️⃣ Landing Page  
📂 Folder: WebDev-L1-LandingPage  

A visually appealing landing page designed to showcase a product or service.  
This project focuses on layout structure, typography, color usage, and call-to-action elements.

Key Features:
- Navigation bar
- Hero section with headline and CTA
- Multiple content sections
- Clean and modern UI design

Local Preview:  
https://thomas-edwin.github.io/OIBSIP/WebDev-L1-LandingPage/

2️⃣ Portfolio Website  
📂 Folder: WebDev-L1-Portfolio  

A personal portfolio website highlighting skills, projects, and contact information.  
Designed to demonstrate professional presentation and personal branding.

Key Features:
- About section
- Skills overview
- Projects showcase
- Contact section
- Responsive layout

Local Preview:  
https://thomas-edwin.github.io/OIBSIP/WebDev-L1-Portfolio/

3️⃣ Temperature Converter  
📂 Folder: WebDev-L1-Temperature  

A simple and interactive temperature converter application.  
This project demonstrates basic logic handling and user input interaction.

Key Features:
- Converts temperature between Celsius, Fahrenheit, and Kelvin
- User-friendly interface
- Real-time conversion output

Local Preview:  
https://thomas-edwin.github.io/OIBSIP/WebDev-L1-Temperature/
