Temperature Converter – Web Development Level 1

This project is a simple and interactive temperature converter application developed as part of the OIBSIP Web Development Level 1 Internship Program. The application allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin through an intuitive interface.

The project demonstrates basic logic handling, user input processing, and real-time output display while maintaining a clean and user-friendly design.

Key Features:
- Converts temperatures between Celsius, Fahrenheit, and Kelvin
- Simple and easy-to-use interface
- Instant conversion results

Technologies Used:
- HTML5
- CSS3

Local Preview:
http://127.0.0.1:5500/WebDev-L1-Temperature/index.html

Purpose:
This project was built to practice basic front-end logic implementation and enhance understanding of user interaction and data handling in web applications.
